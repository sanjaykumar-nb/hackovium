## 🚀 Deployment Instructions (Copy & Paste)

### Step 1: Push to GitHub (5 minutes)

```bash
cd e:/downloads/projects/hackovium

git remote add origin https://github.com/sanjaykumar-nb/hackovium.git
git branch -M main
git push -u origin main
```

**Result:** https://github.com/sanjaykumar-nb/hackovium

### Step 2: Deploy to HF Spaces (10 minutes)

1. Go to https://huggingface.co/spaces
2. Create new Space:
   - Name: "Hackovium"
   - SDK: "Docker"
   - License: "MIT"
3. Set secret:
   - Name: `ANTHROPIC_API_KEY`
   - Value: Your API key
4. Run commands:

```bash
git clone https://huggingface.co/spaces/nbsanjaykumar/Hackovium hf-space
cd hf-space
cp -r ../hackovium/. .
git add -A
git commit -m "🚀 Deploy FUTURE YOU AI to HF Spaces"
git push
```

**Result:** https://huggingface.co/spaces/nbsanjaykumar/Hackovium
