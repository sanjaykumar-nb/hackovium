# FUTURE YOU AI on Hugging Face Spaces 🚀

This application is deployed and running on Hugging Face Spaces!

## How to Deploy to HF Spaces

### Prerequisites
- Hugging Face account
- Hugging Face Space created
- Anthropic API key

### Step 1: Create a Space
1. Go to [huggingface.co/spaces](https://huggingface.co/spaces)
2. Click "Create new Space"
3. Choose "Docker" runtime
4. Give it a name (e.g., "Hackovium")

### Step 2: Set Secrets
In your Space settings:
1. Go to "Settings" → "Repository secrets"
2. Add `ANTHROPIC_API_KEY` with your API key value

### Step 3: Deploy Code
Push your code to the Space's git repo:

```bash
# Clone the space
git clone https://huggingface.co/spaces/YOUR_USERNAME/YOUR_SPACE_NAME Hackovium
cd Hackovium

# Copy all files from this repo
cp -r ../* .

# Push to HF
git add .
git commit -m "Initial deployment"
git push
```

The Space will automatically build and deploy!

### Step 4: Access
Your app will be available at:
```
https://huggingface.co/spaces/YOUR_USERNAME/YOUR_SPACE_NAME
```

## Environment Variables
The Space should have:
- `ANTHROPIC_API_KEY` - Your Anthropic API key (set in secrets)
- `NODE_ENV` - Automatically set to 'production'
- `PORT` - Automatically set to 7860

## Architecture
- **Backend**: Node.js/Express server running on port 7860
- **Frontend**: React app built and served as static files
- **Database**: None (stateless)

## Troubleshooting

### Build fails
- Check that `npm install` works locally
- Verify node version compatibility
- Check that all dependencies are in package.json

### Server won't start
- Verify `ANTHROPIC_API_KEY` is set in secrets
- Check logs in the Space's "Logs" tab
- Ensure port 7860 is being used

### Frontend shows errors
- Clear browser cache
- Check that React build succeeded
- Verify API calls use correct localhost/port

## Local Development

### With Docker Compose
```bash
docker-compose up
```
Visit http://localhost:5000

### Without Docker
```bash
# Terminal 1
cd server && npm install && npm run dev

# Terminal 2
cd client && npm install && npm start
```
Visit http://localhost:3000
